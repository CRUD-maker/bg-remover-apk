# AI Background Remover

A professional desktop application to remove image backgrounds automatically using AI.

## Features
- **Drag & Drop**: Simply drag images into the window.
- **Clipboard Support**: Paste images directly with `Ctrl+V`.
- **AI Powered**: Uses `rembg` (U2Net) for high-quality background removal.
- **Format Support**: Open and save JPG, PNG, WebP.
- **Modern UI**: Dark theme with side-by-side comparison.

## Installation

**No installation required!**
This folder contains a fully portable version of the app.

## How to Run
**Double-click `launch_silent.vbs`**

(This launches the app silently in the background without a black console window).

*Alternate method (Debug):*
If you have issues, run `run_portable.bat` to see error messages.

**First Run Note**: The AI model (~170MB) will download automatically on the first image you process.

## Controls
- **Open Image**: Browse for a file.
- **Save Result**: Save the processed image as PNG with transparency.
- **Reset**: Clear the current workspace.
